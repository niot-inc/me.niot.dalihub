전문가급 DALI 조명을 스마트홈으로!

DALIHub는 상업용 DALI 조명 시스템을 Homey와 연결하여, 직관적인 스마트홈 인터페이스로 전문가급 조명을 제어할 수 있게 해줍니다.

주요 기능:
• DT6, DT7 DALI 조명의 정밀한 밝기 제어 (DT8 색상 제어는 추후 지원 예정!)
• 직접 DALI 레벨 제어 (0-254) - 1% 미만 디밍 가능
• 페이드 타임 제어로 부드러운 전환 효과 (0.7초~90.5초, 16가지 프리셋)
• 씬 컨트롤러로 DALI 씬 호출 (페이드 타임 지원)
• 점유 센서로 자동화 구현 (Lunatone DALI-2 CS로 테스트)
• 조도 센서로 주변 밝기 측정 (Lunatone DALI-2 CS로 테스트)
• 그룹 기능으로 구역별 조명 제어
• 푸시 버튼을 Homey 플로우에 통합 (Lunatone DALI-2 MC로 테스트)
• 모든 장치의 실시간 상태 업데이트

필요 사항:
이 앱을 사용하려면 DALIHub 소프트웨어와 ATX LED의 DALI HAT 하드웨어가 필요합니다.
